# BASICFlow

![GitHub repo size](https://img.shields.io/github/repo-size/digitsensitive/basic-flow)
![GitHub Release](https://img.shields.io/github/v/release/digitsensitive/basic-flow)

BASICFlow is a lightweight BASIC programming environment written in `C` and
`SDL3`. This project was created for educational purposes.

<img src="./readme_image.png" alt="BASICFlow" style="width: 100%;"/>

## Features

1. **Cross-Platform Compatibility**: BASICFlow is designed to work seamlessly
   across various platforms, including Windows, macOS, and Linux.
2. **Open Source**: BASICFlow is an open-source project, encouraging collaboration
   and contributions from the community. Feel free to fork the repository,
   report issues, or submit pull requests.

## How to build

Initialize submodules:

```bash
git submodule update --init --recursive
```

Build Release:

```bash
cd build
cmake --build . --config Release
```
